# Описание кода:

## Последовательная версия:

### Простая реализация цикла для вычисления суммы квадратов чисел от 1 до N.
###	Замер времени выполнения с использованием ```Stopwatch```.

## Параллельная версия с блокировкой (lock):

### Распараллеливание с использованием Parallel.For.
### Использование lock для синхронизации доступа к общей переменной sumParallel.

## Параллельная версия без использования блокировки внутри цикла:

### Используются локальные переменные для каждого потока.
### Итоговая сумма вычисляется путем объединения локальных результатов с минимизацией использования блокировок.
